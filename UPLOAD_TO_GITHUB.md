# Upload This App To GitHub

Repo:

```txt
https://github.com/1ofdakoolkidz/Ai-council
```

## Easy Web Upload

1. Open the repo link above.
2. Click **Add file**.
3. Click **Upload files**.
4. Open this folder on your computer:

```txt
C:\Users\AEHol\Documents\Codex\2026-06-01\i-want-to-be-able-to
```

5. Drag these files into GitHub:

```txt
.env.example
.gitignore
app.js
DEPLOY.md
index.html
package.json
README.md
render.yaml
server.js
styles.css
UPLOAD_TO_GITHUB.md
```

6. Do **not** upload a file named `.env`.
7. Scroll down.
8. In the box that says commit message, type:

```txt
Initial AI Council app
```

9. Click **Commit changes**.

## Important

Never upload your real `.env` file.

`.env` is where private API keys go. GitHub should only get `.env.example`.
